# Chicken Inn Zimbabwe - LUV DAT Chicken 🇿🇼

## Project Overview

This is a **learning project** created by **Tonde** to demonstrate modern web development skills. It's a complete redesign of the Chicken Inn Zimbabwe website, built to surpass industry standards including KFC's website design.

**Purpose**: Educational and portfolio demonstration  
**Developer**: Tonde  
**Status**: Learning Project - Not affiliated with Chicken Inn Zimbabwe

## Features

- 🍗 **Complete Online Ordering System** - Order chicken from anywhere in Zimbabwe
- 🎠 **Hero Carousel** - Beautiful rotating display of chicken and food items
- 📱 **Responsive Design** - Optimized for all devices
- 🛒 **Advanced Shopping Cart** - Full cart management with quantity controls
- 📍 **Nationwide Delivery** - Coverage across all Zimbabwe provinces
- 🎨 **Premium UI/UX** - Designed to exceed KFC website standards
- ⚡ **High Performance** - Built with modern React and Vite

## Technologies Used

- **Frontend**: React 18.3.1 with TypeScript
- **Build Tool**: Vite 5.4.19
- **Styling**: Tailwind CSS 3.4.17 with custom design system
- **UI Components**: shadcn/ui with Radix UI primitives
- **State Management**: React Query (@tanstack/react-query)
- **Routing**: React Router DOM
- **Icons**: Lucide React
- **Animations**: Custom CSS animations with Tailwind
- **Forms**: React Hook Form with Zod validation

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone <repository-url>

# Navigate to project directory
cd chickenin

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
